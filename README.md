# UnityGame

# Unity Game Preview

A 2D platformer game built using Unity and C#.  
This project demonstrates gameplay systems, prefab management, scene organization, and player customization mechanics.

---

## 🎮 Features

- Character selection system
- Multiple playable levels
- Enemy AI (Snail, Beetle)
- Prefab-based UI system
- Scene transitions (Main Menu → Levels)
- Player skin selection system
- Organized prefab structure
- Clean Unity project structure

---

## 🛠 Tech Stack

- Unity Engine
- C#
- Prefab System
- Scene Management
- Git Version Control

---

## 🕹 Controls

- Move: A / D or Arrow Keys
- Jump: Space
- Select Character: UI Buttons

---

## 📂 Project Structure

- `Assets/` → Game assets, scripts, scenes, prefabs
- `Packages/` → Unity package dependencies
- `ProjectSettings/` → Unity project configuration
